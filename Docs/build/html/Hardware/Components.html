Components¶

Fault Indicator/Strobe Light¶

The strobelight contains 4, 3 watt LED’s and is powered by a 12V line. It has 4 wires for setup although we only need to use 3. Outside of power and ground, the other 2 wires are for sync and pattern. The sync wire is if we are using multiple strobelights that must strobe at the same time. The pattern wire must be shorted with power if the pattern is to be changed. Once it is set to a pattern, it remembers the pattern with on board flash memory.
